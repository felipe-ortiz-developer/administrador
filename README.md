# Administrador
Proyecto de prueba para poner en practica conceptos de angular 16.
![Alt text](image.png)

#### Funcionalidades
- Listar Empleados
- Agregar Empleado
- Editar Empleado
- Agregar Caracteristica al Empleado
#### Conceptos puestos en practica: 
- Interpolación
- Directivas 
    - Formulario con [(ngModel)], [ngClass], [ngStyle] y event binding
    - Lista con *ngFor
- Estructura de proyecto componentes, modelos y servicios
    - Clases Modelo
        - Empleado
        - Caracteristicas
    - Comunicacion entre componentes
        - @Input
        - @Output
        - OnInit
    - Services
    - Objetos anonimos
- Integración:
    - Bootstrap 5
    - Angular-font-awesome
    - ng-bootstrap
